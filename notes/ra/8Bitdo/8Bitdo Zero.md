# 8Bitdo Zero

| function     | button                  | LED                  |
| ------------ | ----------------------- | -------------------- |
| Power on     | Press and hold 2s START | blink once per cycle |
| Pairing mode | Press & hold 2s SELECT  | rapidly blink        |
| Paired       | N.A.                    | solid blue           |
