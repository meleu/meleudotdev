# 8Bitdo NES30

## Bluetooth Connection

| function     | button                  | LED                  |
| ------------ | ----------------------- | -------------------- |
| Power on     | Press and hold 1s START | blink once per cycle |
| Pairing mode | Press & hold 3s SELECT  | rapidly blink        |
| Paired       | N.A.                    | solid blue           |
